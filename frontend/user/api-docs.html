Product Endpoints

GET /api/produk API Key

Mendapatkan semua data produk buah & sayur

Headers

x-api-key: YOUR_API_KEY

Response Example

[ { "id": 1, "nama": "Apel Fuji", "kategori": "Buah", "harga": "25000", "stok": 100, "foto": "https://..." } ]
GET /api/produk/search API Key

Mencari produk berdasarkan nama dan/atau kategori

Query Parameters

nama Nama produk (opsional)

kategori "Buah" atau "Sayur" (opsional)

Example Request

GET /api/produk/search?nama=apel&kategori=Buah x-api-key: YOUR_API_KEY
