Manajemen Produk

GET /admin/produk Protected

Mendapatkan semua data produk (Admin only)

Headers

Authorization: Bearer YOUR_JWT_TOKEN

Response Example

[ { "id": 1, "nama": "Apel Fuji", "kategori": "Buah", "harga": "25000", "stok": 100, "foto": "https://..." } ]
GET /admin/produk/search Protected

Mencari produk berdasarkan nama/kategori

Headers

Authorization: Bearer YOUR_JWT_TOKEN
POST /admin/produk Protected

Menambahkan produk baru

Request Body

{ "nama": "Jeruk Mandarin", "kategori": "Buah", "harga": "12000", "stok": 50, "foto": "jeruk.jpg" }

Response

{ "message": "Produk berhasil ditambahkan" }
PUT /admin/produk/:id Protected

Update data produk berdasarkan ID

Request Body

{ "nama": "Apel Merah", "harga": "28000", "stok": 75 }

Response

{ "message": "Produk berhasil diupdate" }
DELETE /admin/produk/:id Protected

Menghapus produk berdasarkan ID

Response

{ "message": "Produk berhasil dihapus" }

Manajemen API Key

GET /admin/apikeys Protected

Mendapatkan semua API key yang terdaftar

Response Example

[ { "id": 1, "nama": "User Name", "api_key": "uuid-key", "status": "aktif", "limit_daily": 1000, "used_daily": 45 } ]
PUT /admin/apikeys/:id/nonaktif Protected

Menonaktifkan API key user

Response

{ "message": "API Key berhasil dinonaktifkan" }
GET /admin/apikeys/{id}/statistik Protected

Menampilkan statistik penggunaan API key

Response

"{ "api_key": "UUID", "limit_daily": 500, "used_daily": 20, "last_used": "2026-01-13 10:00:00" };
PUT /admin/apikeys/:id/limit Protected

Mengubah limit harian API key

Request Body

{ "limit_daily": 2000 }

Response

{ "message": "Limit berhasil diupdate" }
